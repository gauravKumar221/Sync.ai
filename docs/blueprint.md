# **App Name**: LeadFlowAI

## Core Features:

- Lead Capture Form: Capture lead data (name, phone, message, source) from the landing page form.
- WhatsApp Lead Simulation: Simulate lead creation and auto-reply trigger upon receiving a WhatsApp message, storing lead data.
- AI-Powered CRM Dashboard: Display and manage leads, track progress, and view finance summaries. Also features top stats cards like total leads and new leads today.
- Conversation Panel: Display a chat-like UI to view customer and agent messages with timestamps.
- Automated Responses: The LLM uses a tool to trigger pre-defined auto-replies based on message content and lead source, enhancing initial customer engagement.
- Analytics Dashboard: Visualize leads by source, conversion rates, and agent performance using charts and tables.
- Appointment Calendar: Schedule and manage appointments, including client details, date, time, and assigned agent.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Dark gray (#212121) for a modern dark mode dashboard.
- Accent color: Vibrant red (#FF5722) for highlighting CTAs and important stats, creating a Red & Blue gradient feel.
- Body and headline font: 'Inter', sans-serif, for a clean and modern user interface. Suitable for both headlines and body text.
- Use a consistent set of simple, professional icons from a library like Font Awesome or Remix Icon.
- Use a grid layout with rounded cards and soft shadows for a clean, modern SaaS look.
- Apply subtle hover and transition animations to buttons and cards for enhanced user interaction.
